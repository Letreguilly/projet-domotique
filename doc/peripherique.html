Les périphériques sont composé d'un chassis de base, il en existe deux type différents : les capteurs et les acctionneurs. Chaque périphérique posséde une interface réseau tcp/ip pour communiquer avec les autres

le micro programme des périphériques doit contenir à minima quatre paramétres :

  • son nom
  • son IP
  • sa MAC address
  • L'addresse IP ou le nom du serveur(si la résolution de nom dns est implémentée)
